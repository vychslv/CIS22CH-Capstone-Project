Place OpenFlights CSV data files here:
  - airports.dat
  - airlines.dat
  - routes.dat

sample_destinations.csv - Example file for "Upload destinations" (format: name,city,country,iata,lat,lon).

If you see only one suggestion or wrong text in search boxes, the .dat files
are not valid CSV. Replace them:

  1. Run download_data.bat in this folder (double-click or from command line).
  2. Or download the raw CSV files manually (save as plain text, not HTML):
     https://raw.githubusercontent.com/jpatokal/openflights/master/data/airports.dat
     https://raw.githubusercontent.com/jpatokal/openflights/master/data/airlines.dat
     https://raw.githubusercontent.com/jpatokal/openflights/master/data/routes.dat
